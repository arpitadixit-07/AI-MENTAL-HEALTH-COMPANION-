# AI Mental Health Companion Advanced

This is an advanced AI-powered mental health companion app with visuals, audio, and interactive tools.

## Features
- Sentiment Analysis (positive/negative/neutral)
- Motivational Quotes with Images
- Daily Wellness Tips
- Chat History
- Mood Trend Graph + Pie Chart
- Voice Output (Text-to-Speech)
- Guided Meditation Video
- Breathing Exercise Animation
- Relaxing Music

## How to Run
1. Extract the ZIP file.
2. Open terminal in the folder.
3. Install requirements:
   ```bash
   pip install -r requirements.txt
   ```
4. Run the app:
   ```bash
   streamlit run ai_mental_health_companion_advanced.py
   ```
5. The app will open in your browser.
